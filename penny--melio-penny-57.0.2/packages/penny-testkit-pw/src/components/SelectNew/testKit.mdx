# SelectNew Playwright Test Kit
---
Testing utilities for the SelectNew component that provide easy access to interactions, states, and component behavior for Playwright testing.

## Import

```typescript
import { createSelectNewTestKit } from '@melio/penny-testkit-pw';
```

## Methods

<Markdown>
  {`
| Method | Description | Return value |
|--------|-------------|--------------|
| \`dataTestId\` | The data-testid used to identify the component | \`string\` |
| \`getElement()\` | Direct access to the DOM element | \`Locator\` |
| \`clickTrigger()\` | Clicks on the trigger input to open the select | \`Promise<void>\` |
| \`clickOptionByText(text: string)\` | Clicks an option by its visible text | \`Promise<void>\` |
| \`clickOptionByTestId(testId: string)\` | Clicks an option by its data-testid | \`Promise<void>\` |
| \`clickOptionByIndex(index: number)\` | Clicks an option by its index in the list | \`Promise<void>\` |
| \`typeSearch(text: string)\` | Types text into the search input | \`Promise<void>\` |
| \`clickSearchButton()\` | Clicks on the search button | \`Promise<void>\` |
| \`clickOutside()\` | Clicks outside the select to close it | \`Promise<void>\` |
| \`pressEsc()\` | Presses the Escape key to close the select | \`Promise<void>\` |
| \`clickClearSearch()\` | Clears the search input | \`Promise<void>\` |
| \`getValue()\` | Gets the selected value | \`Promise<string \| null>\` |
| \`getIsDisabled()\` | Checks if the select is disabled | \`Promise<boolean>\` |
| \`getIsInvalid()\` | Checks if the select is invalid | \`Promise<boolean>\` |
| \`getIsViewMode()\` | Checks if the select is in view mode | \`Promise<boolean>\` |
| \`getIsReadOnly()\` | Checks if the select is read-only | \`Promise<boolean>\` |
| \`getIsRequired()\` | Checks if the select is required | \`Promise<boolean>\` |
| \`clickClearButton()\` | Clicks on the clear button | \`Promise<void>\` |
| \`getIsMenuOpen()\` | Checks if the select menu is currently open | \`Promise<boolean>\` |
`}
</Markdown>

## Examples

```typescript
import { test, expect } from '@playwright/test';
import { createSelectNewTestKit } from '@melio/penny-testkit-pw';

test.describe('SelectNew Component', () => {
  test('should check if the select is disabled', async ({ page }) => {
    const testKit = createSelectNewTestKit(page);
    
    await page.goto('/select-new-disabled-story');

    expect(await testKit.getIsDisabled()).toBe(true);
  });

  test('should select an option', async ({ page }) => {
    const testKit = createSelectNewTestKit(page);
    
    await page.goto('/select-new-story');

    await testKit.clickTrigger();
    await testKit.selectOptionByText('New York');

    await expect(page.getByRole('listbox')).not.toBeVisible();
    expect(await testKit.getValue()).toBe('new_york');
  });

  test('should clear the select', async ({ page }) => {
    const testKit = createSelectNewTestKit(page);
    
    await page.goto('/select-new-story');

    await testKit.clickTrigger();
    await testKit.selectOptionByText('Boston');

    expect(await testKit.getValue()).toBe('boston');

    await testKit.clearSelect();

    expect(await testKit.getValue()).toBeNull();
  });
});
```
