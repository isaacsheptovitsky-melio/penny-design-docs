# VerificationCodeField Playwright Test Kit
---

Testing utilities for the `VerificationCodeField` component that provide easy access to component interactions, states, and accessibility features for Playwright testing.

## Import

```typescript
import { createVerificationCodeFieldTestKit } from '@melio/penny-testkit-pw';
```

## Methods

<Markdown>
  {`
| Method | Description | Return value |
|--------|-------------|--------------|
| \`dataTestId\` | The data-testid used to identify the component | \`string\` |
| \`getElement()\` | Direct access to the DOM element | \`Locator\` |
| \`getValue()\` | Gets the current value of the field | \`Promise<string>\` |
| \`getIsInvalid()\` | Checks if the field is invalid | \`Promise<boolean>\` |
| \`getErrorMessage()\` | Gets the error message when field is invalid | \`Promise<string \| null>\` |
| \`type(code)\` | Types code into the field | \`Promise<void>\` |
| \`clear()\` | Clear the field | \`Promise<void>\` |
`}
</Markdown>

## Examples

```typescript
import { test, expect } from '@playwright/test';
import { createVerificationCodeFieldTestKit } from '@melio/penny-testkit-pw';

test.describe('VerificationCodeField Component', () => {
  test('should show value', async ({ page }) => {
    const testKit = createVerificationCodeFieldTestKit(page, { dataTestId: 'custom-datatestid' });
    await page.goto('/some-page');

    await testKit.type('1234');

    expect(await testKit.getValue()).toBe('1234');
  });

  test('should be invalid', async ({ page }) => {
    const testKit = createVerificationCodeFieldTestKit(page, { dataTestId: 'custom-datatestid' });
    await page.goto('/some-page');
    
    await testKit.type('4321');

    expect(await testKit.getIsInvalid()).toBeTruthy();
    expect(await testKit.getErrorMessage()).toBe('Invalid code');
  });

  test('should clear the field', async ({ page }) => {
    const testKit = createVerificationCodeFieldTestKit(page, { dataTestId: 'custom-datatestid' });
    await page.goto('/some-page');

    await testKit.type('123');
    expect(await testKit.getValue()).toBe('123');

    await testKit.clear();
    expect(await testKit.getValue()).toBe('');
  });
});
```
