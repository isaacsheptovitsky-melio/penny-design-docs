<Title>useLinearProgressInitialAnimation</Title>

The `useLinearProgressInitialAnimation` hook provides a simple way to animate a `LinearProgress` value from an initial value to a target value. This is particularly useful for creating smooth initial animations when a progress bar first appears.

## Usage Example

```tsx
import { useLinearProgressInitialAnimation } from '@melio/penny';
import { LinearProgressIndicator, LinearProgressRoot, LinearProgressTrack } from '@melio/penny';

const Example = () => {
  const { value } = useLinearProgressInitialAnimation({ 
    initialValue: 0, 
    targetValue: 60 
  });

  return (
    <LinearProgressRoot value={value}>
      <LinearProgressTrack>
        <LinearProgressIndicator />
      </LinearProgressTrack>
    </LinearProgressRoot>
  );
};
```

You can see this example in action in the [With Initial Animation story](/story/feedback-components-linear-progress--with-initial-animation).

## Notes

- The hook uses `useState` and `useEffect` internally to manage the animation
- The value transitions from `initialValue` to `targetValue` after the specified `delay`
- The timer is automatically cleaned up when the component unmounts or when dependencies change
