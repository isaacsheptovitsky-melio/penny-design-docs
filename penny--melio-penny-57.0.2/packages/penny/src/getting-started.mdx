# Penny Utils package

---

Welcome to the **Penny Utils** package! This package contains a set of utility functions and helpers designed to streamline your development process and enhance the consistency.

## Installation

Run one of the following commands to add **Penny Utils** to your project (currently we support only canary versions):

```bash
yarn add @melio/penny-utils@canary
# or
npm install @melio/penny-utils@canary
```
