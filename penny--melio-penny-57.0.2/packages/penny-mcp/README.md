# Penny MCP

Model Context Protocol (MCP) server for the Penny Design System. Enables LLMs and AI tools to discover and use Penny components and design tokens.

## Features

- **List Components:**  
  `list_penny_components` — Get all Penny components, optionally filtered by category.
- **Component Details:**  
  `get_penny_component_details` — Get full API, props, and usage for any component.
- **List Design Tokens:**  
  `list_penny_design_tokens` — Get all Penny design tokens, optionally filtered by type.

## Installation

```
mcpm add penny-mcp
```
