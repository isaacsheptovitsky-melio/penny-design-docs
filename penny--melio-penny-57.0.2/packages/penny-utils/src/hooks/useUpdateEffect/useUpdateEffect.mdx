# `useUpdateEffect`

The `useUpdateEffect` hook works like `useEffect`, but skips the effect on the initial render. It is useful for scenarios where you want to perform side effects only after the initial render.

## Usage

```tsx
import { useUpdateEffect } from '@melio/penny-utils';

const ExampleComponent = () => {
  const [count, setCount] = useState(0);

  useUpdateEffect(() => {
    console.log('Effect runs on updates, not on the initial render.');
  }, [count]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
};
