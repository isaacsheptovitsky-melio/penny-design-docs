## Usage

- **Show how many items** belong to a category (e.g. "Invoices (4)").
- **Indicate filtered or selected** item counts.
- **Display totals** in tabs, pills, or section headers.

<Callout variant="tip">Pair counters with a clear label to avoid confusion — context is key.</Callout>

## Variants

### Statuses

Six semantic statuses are available. Choose the color that matches the meaning of the count.

<Story of={CounterStories.Statuses} />

### Large number

Values above 99 are rendered as `+99`, keeping the counter at a consistent size.

<Story of={CounterStories.LargeNumber} />

## Do's and don'ts

<Do>pair the counter with a clear label or UI element (e.g. "Archived (2)") so the number has context.</Do>

<Dont>use a counter for critical alerts — it communicates quantity, not urgency. Reach for a Section Banner or status messaging when the user must act.</Dont>
