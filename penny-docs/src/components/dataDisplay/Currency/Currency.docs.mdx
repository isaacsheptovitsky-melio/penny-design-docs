## Usage

- Display monetary amounts consistently — the value is formatted with the currency symbol, grouped integers, and a de-emphasized fraction.
- Set `currency` to the relevant ISO code; the symbol and formatting follow it.

## Sizes

Two sizes are available — `small` (default) and `large`.

<Canvas of={CurrencyStories.Sizes} />

## Inverse

Use the `inverse` variant on dark or colored surfaces.

<Canvas of={CurrencyStories.Inverse} />

## Currencies

The component formats any ISO currency code.

<Canvas of={CurrencyStories.Currencies} />

<Callout variant="note">Formatting uses the browser's `Intl` number formatter with a narrow currency symbol.</Callout>
