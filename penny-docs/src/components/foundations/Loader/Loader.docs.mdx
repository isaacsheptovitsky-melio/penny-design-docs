## Usage

Use the Loader when:

- A page or section is awaiting data.
- A user-initiated action triggers a delay (submitting a form, fetching data).
- It's important to reassure users that the system is actively working.

## Colors

Set the `color` prop to any Penny color token. By default the Loader uses the brand color.

<Canvas of={LoaderStories.Colors} />

<Callout variant="note">By default the Loader exposes an `aria-label` of "Loading…". Override it with `loadingText` to announce a custom message.</Callout>

## Use cases

The Loader is used in two scenarios: when a screen is waiting for data to load, and when an asynchronous action is in progress.

### Screen data loading

Use the Loader to fill the screen while the page waits for data from the server.

<Canvas of={LoaderStories.UseCaseScreenLoading} />

### Async action

Saving or paying a bill is an asynchronous action that requires a server response. Show the Loader while waiting for that confirmation.
