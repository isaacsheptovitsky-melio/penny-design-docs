## Usage

- **Apply spacing tokens** for margins, paddings, layout gaps, and grid gutters.
- **Avoid arbitrary pixel values** unless a special case justifies it.
- Use **larger tokens** to separate sections and create breathing room.
- Use **smaller tokens** inside compact components or dense UI areas.
