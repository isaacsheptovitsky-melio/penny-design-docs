## Breakpoint tokens

Each breakpoint is a min-width. Responsive values resolve at the breakpoint that's currently active, so styles set at a smaller key apply until the next key is reached.

## Container width

A container's `max-width` controls how wide it can grow before capping — it expands to fill available space but stops at the specified limit. Penny defines five container width variants; choose based on the task density and layout complexity of the screen.
