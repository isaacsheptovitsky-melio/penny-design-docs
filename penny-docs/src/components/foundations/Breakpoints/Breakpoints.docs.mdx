## Breakpoint tokens

Each breakpoint is a min-width. Responsive values resolve at the breakpoint that's currently active, so styles set at a smaller key apply until the next key is reached.

<Canvas of={BreakpointsStories.BreakpointTokens} />

<Callout variant="note">Components read these via Chakra's responsive value syntax (objects keyed by breakpoint, e.g. `xs` → `m`) and the `useBreakpointValue` / `useBreakpoint` hooks.</Callout>
