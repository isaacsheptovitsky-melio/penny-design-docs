## Token explorer

Browse the radius tokens across the three levels. **Semantic** tokens set the default radius for a whole component category; **component** tokens pin a specific component part.

<Canvas of={BorderRadiusStories.TokenExplorer} />

- **`border-radius.action.default`** (semantic) → drives **action** components: Button, Icon Button, Date picker.
- **`border-radius.input.default`** (semantic) → drives **input** components: Text field, Select, Search bar, Combobox, and the rest of Selection & Inputs.
- **`border-radius.pill.default`** (component) → the **Pill** component's fully-rounded corners.

<Callout variant="note">Always reach for the highest-level token available — component or semantic over a raw global value — so theming stays consistent.</Callout>
