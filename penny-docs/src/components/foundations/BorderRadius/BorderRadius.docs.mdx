## Global tokens

The base radius scale applied across components for consistent corners.

<Canvas of={BorderRadiusStories.GlobalTokens} />

## Semantic tokens

Semantic tokens define the default radius for a category of components — reach for these in component themes rather than a global value.

- **`border-radius.action.default`** → `border-radius.200` (8px) — most action components (Button, Icon Button, Date picker).
- **`border-radius.input.default`** → `border-radius.200` (8px) — most input components (Text field, Select, Search bar, …).

## Component tokens

- **`border-radius.pill.default`** → `border-radius.full` — the Pill component's fully-rounded corners.

<Callout variant="note">Always reach for the highest-level token available — component or semantic over a raw global value — so theming stays consistent.</Callout>
