## Sizes

Three sizes are available — `extra-small` (12px), `small` (16px), and `large` (24px).

<Canvas of={IconStories.Sizes} />

## Grid & sizes

Each icon has two versions — on a **16px** and on a **24px** grid.

<Canvas of={IconStories.GridSizes} />

### Large: 24px grid

On the large 24px grid, the stroke width is **2px** with rounded ends, corners are usually **2px**, and there is a **1px** margin (frame).

<Canvas of={IconStories.LargeGrid} />

### Small: 16px grid

On the small 16px grid, the stroke width is **1.5px** with rounded ends, corners are usually **1.5px**, and there is a **1px** margin (frame). On the 16px grid, align the outer edge of shapes to pixel boundaries.

## Adding icons to Penny

New icons follow a specific Figma layer structure so Penny can consume them correctly.

<Canvas of={IconStories.FigmaStructure} />

- **Outer frame** — named after the icon, set in Auto Layout with Width and Height set to Hug
- **Inner frame (`wrapper`)** — fixed at either 16×16 or 24×24, depending on the icon size
- **The icon (`Vector`)** — SVG in a single flat layer

For the full process, see [How to add a new icon to Penny](https://www.notion.so/meliopayments/How-to-add-a-new-icon-to-Penny-16c66d69640a807580e2d0eaaf0f6d33).

## Accessibility

When choosing or naming an icon, focus on **what it does**, not what it looks like.

<Dont>name icons after their visual form — "Trash", "Pencil", "Cross", "Tick", "Magnifier".</Dont>

<Do>name icons after their function — "Delete", "Edit", "Close", "Check", "Search".</Do>

<Callout variant="note">Ask yourself: *If I couldn't see the icon, would the name still help me understand what it does?*</Callout>
