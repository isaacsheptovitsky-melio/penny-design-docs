## Global tokens

The base border-weight scale. Tokens pair a width with the `solid` style.

<Canvas of={BordersStories.GlobalTokens} />

## Semantic tokens

Semantic tokens map a category to a global weight — use these in component themes.

- **`border.action.default`**, **`border.container.default`**, **`border.input.default`** → `global.25` (1px) — the standard component border.
- **`border.focus`** → `global.50` (2px) — the focus ring.

<Callout variant="note">Border color is set separately via color tokens (e.g. `semantic.border.static`); these tokens define the **weight and style** only.</Callout>
