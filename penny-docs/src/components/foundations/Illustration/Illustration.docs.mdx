## Sizes

Three sizes are available — `small` (96px), `medium` (120px), and `large` (144px).

For partner implementations, the two brand tokens (`illustration-brand-primary` and `illustration-brand-secondary`) can be overridden to match the partner's color palette, while the border and background remain fixed.

## Adding illustrations to Penny

New illustrations must follow the Penny SVG structure and use the `penny-illustration-*` CSS classes so the token system can theme them correctly. See the **How to add an illustration** link above for the full process.
