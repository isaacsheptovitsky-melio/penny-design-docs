## Token architecture

Penny uses a three-level token hierarchy — each level builds on the one below it. Never skip a level when applying color in a component theme.

<Canvas of={ColorTokensStories.TokenArchitecture} />

<Callout variant="note">

Always reach for the highest-level token available. Use **semantic** tokens in component themes rather than hard-coding global values — this is what makes white-label theming possible.

</Callout>
