## Usage

Use a link when you want to:

- **Navigate to a different page within the Melio app** — e.g. from the dashboard to the "Payables" page.
- **Direct users to an external URL** — such as a help article or a partner website.

<Callout variant="tip">To prevent user confusion, always make sure link text clearly describes its destination. For example, use "View invoice details" instead of a generic "Click here."</Callout>

## Variants

### Type

Links come in two main types:

- **`standalone`** — used for navigation that stands on its own, such as in a navigation menu or as a primary call to action. It has more styling options like `size` and `isBold`.
- **`inline`** — used within a sentence or block of text to link specific words without disrupting the flow of the content. For example: "Please review the Terms of Service before proceeding."

### Color

You can apply different colors to a link to fit its context:

<Story of={LinkStories.Colors} />

### Size

Standalone links can be set to two sizes.

<Canvas of={LinkStories.Sizes} />

- **`medium`** — the default size for most standalone links (Body3 style).
- **`large`** — used for more prominent navigational links that need to draw more attention (Body2 style).

### Bold

Add `isBold` to a `standalone` link to give it more visual weight.

<Canvas of={LinkStories.BoldLinks} />

### Disabled

A link with `isDisabled` is not clickable and appears grayed out.

<Canvas of={LinkStories.DisabledLinks} />

<Callout variant="warning">

Disabled links still render a visible element. If the link should not be available at all in the current context, hide it rather than disabling it — a hidden absent option is less confusing than a visible but unavailable one.

</Callout>

### Ellipsis

Use the `shouldSupportEllipsis` style when the link text might be too long for its container. It truncates the text and adds "…" to the end.

<Callout variant="warning">Truncated links can pose an accessibility challenge. Prioritize clear and concise link labels that fit the available space whenever possible.</Callout>
