## Usage

- Use when multiple components should appear as a unified section.
- Adjust **spacing** with design tokens to control the rhythm between items.
- Use **variant** to switch between horizontal and vertical layout.
- Use **hasDivider** for visual separation between elements.

## Variants

### Direction

Horizontal (default) lays items out in a row; vertical stacks them in a column.

<Canvas of={GroupStories.Direction} />

### With dividers

Turn dividers on for visual separation between items.

<Canvas of={GroupStories.WithDividers} />

### Horizontal scrolling

`allowOverflowX` lets content scroll horizontally — useful for dynamic lists.

<Canvas of={GroupStories.HorizontalScroll} />

<Callout variant="warning">Horizontal scrolling can be challenging for accessibility — avoid it when possible.</Callout>
